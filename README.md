# Mooch Saloon — Official Website

Premium luxury barber & grooming lounge website for Mooch Saloon, Pushkar, Rajasthan.

**Stack:** React 19 + TypeScript + Vite + Tailwind CSS v4 + Framer Motion

---

## Local Development

```bash
npm install
npm run dev
```
Open http://localhost:3000

## Build for Production

```bash
npm run build
```
Output goes to `dist/` folder.
